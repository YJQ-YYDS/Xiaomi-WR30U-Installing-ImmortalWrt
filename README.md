# Xiaomi-WR30U-Installing-ImmortalWrt
小米 WR30U 刷入 ImmortalWrt 固件完整指南

本指南将详细介绍如何为小米 WR30U 路由器刷入基于 ImmortalWrt 的第三方固件，并开启代理、L2TP 等高级功能。
本固件基于 112M 分区布局，已集成常用依赖，刷入后可直接安装 luci-app-passwall 等插件，实现科学上网、VPN 客户端等需求。

📌 适用设备
小米 WR30U（已刷入第三方 Uboot，分区布局为 112M UBI）

若不确定分区布局，可通过 SSH 查看：cat /proc/mtd，应包含 ubi 和 ubi_kernel，且环境变量中显示 mtd_layout_label=immortalwrt-112m。

🔧 准备工作

1.固件下载
下载：https://wwpp.lanzoum.com/iTENz2z06jkd

2.硬件连接
电脑网线连接路由器 LAN 口
断开路由器 WAN 口连接（避免干扰）

3.电脑 IP 设置
手动设置静态 IP：192.168.31.100，子网掩码 255.255.255.0，网关留空

🚀 刷机步骤（通过 Uboot）
1. 进入 Uboot 恢复模式
路由器断电
用卡针或笔尖按住路由器背面的 Reset 孔
保持按住的同时接通电源
持续按住约 5~10 秒后松开
在浏览器中访问 http://192.168.31.1，应能看到 Uboot 的 Web 界面（英文，有文件上传按钮）

2. 刷写固件
点击“选择文件”或“Choose File”，选中下载的 .bin 固件文件
点击“Update”或“Flash”按钮
等待上传及刷写完成（约 2~3 分钟），期间切勿断电
刷写成功后路由器自动重启，指示灯恢复正常

4. 恢复电脑网络
将电脑 IP 改回“自动获取 IP 地址”

⚙️ 初次配置

1. 登录系统
浏览器访问 http://192.168.31.1（部分固件默认可能是 192.168.1.1，根据实际访问）
用户名：root，密码：无（首次登录后建议设置密码）

3. 基础网络设置
进入“网络” → “接口”，配置 WAN 口（PPPoE 拨号或 DHCP 客户端）
设置 WiFi（网络 → 无线）：编辑 2.4G 和 5G 信号，设置 SSID 和密码

3. 安装代理插件（Passwall）
系统-软件包-更新仓库-搜索Passwall下载软件包即可
